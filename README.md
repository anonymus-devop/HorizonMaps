HorizonMaps es una aplicación moderna de navegación inteligente con diseño tipo iOS 26 Liquid Glass UI, animaciones fluidas y soporte completo para dispositivos móviles, PWA y futuras integraciones nativas (Capacitor/Kotlin). Combina un mapa interactivo con funciones avanzadas impulsadas por IA para ofrecer una experiencia de exploración clara, accesible y en tiempo real.

La app permite buscar lugares mediante texto o voz estilo asistente inteligente, mostrar rutas optimizadas, navegar con seguimiento en vivo y gestionar puntos de interés con una interfaz rápida, táctil y minimalista. Incluye un botón de centrar ubicación, navegación paso a paso, vista de ruta tipo “love view” al estilo Google Maps, y compatibilidad con TalkBack/VoiceOver para accesibilidad total. Además, soporta modos ligeros, animaciones con Framer Motion y una estructura preparada para integrarse con servicios externos como TTS avanzado, Android Auto, y navegación en tiempo real.
